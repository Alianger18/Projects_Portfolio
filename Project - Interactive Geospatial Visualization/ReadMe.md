## Installation

Start by installing project's requirements
```shell 
pip install -r requirements.txt
```

Then, launch the jupyter notebook 
```Shell 
jupyter notebook notebooks/notebook.ipynb
```

Finally, launch the dashboard locally
```shell
python main.py
```
